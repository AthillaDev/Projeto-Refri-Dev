# 🍹 Projeto Refri do Dev 

Uma aplicação web interativa desenvolvida para praticar manipulação do DOM, animações CSS e lógica de navegação com JavaScript.

O projeto apresenta um carrossel de refrigerantes com transições suaves, indicadores visuais e navegação dinâmica entre os produtos.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript ES6+
- Git
- GitHub
- Vercel

## 🎯 Funcionalidades

- Navegação entre os produtos
- Botões Anterior e Próximo
- Indicadores de slide ativos
- Animações CSS
- Layout responsivo
- Animações com Keyframes
- Manipulação do DOM com JavaScript

 ## 💡 Conceitos Praticados

- Seletores DOM
- Eventos de clique
- Manipulação de Classes
- Arrays e índices
- Animações com CSS
- Posicionamento com Flexbox
- Responsividade
- Acessibilidade com ARIA Labels
